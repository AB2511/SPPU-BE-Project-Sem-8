## 🧪 Assignments Overview

### 🔹 Assignment 1: Boston Housing Price Prediction

* Implemented Linear Regression using Deep Neural Network
* Dataset: Boston Housing (`HousingData.csv`)
* Techniques used:

  * Data preprocessing
  * Feature scaling
  * Neural network regression

**Output:**

* Mean Squared Error (MSE): ~25
* Mean Absolute Error (MAE): ~3.8

---

### 🔹 Assignment 2: IMDB Sentiment Classification

* Binary classification of movie reviews (positive/negative)
* Dataset: IMDB (Keras dataset)
* Techniques used:

  * Text vectorization
  * Deep Neural Network (DNN)

**Output:**

* Accuracy: ~86%

---

### 🔹 Assignment 3: CNN for Fashion MNIST

* Image classification using Convolutional Neural Network
* Dataset: Fashion MNIST
* Techniques used:

  * Convolution layers
  * Max pooling
  * Fully connected layers

**Output:**

* Accuracy: ~82%
