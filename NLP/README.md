## 🧪 Assignments Overview

### 🔹 Assignment 1: Tokenization, Stemming & Lemmatization

* Implemented various tokenization techniques:

  * Whitespace
  * Punctuation-based
  * Treebank
  * Tweet Tokenizer
  * MWE Tokenizer
* Applied:

  * Porter Stemmer
  * Snowball Stemmer
  * Lemmatization (NLTK)

---

### 🔹 Assignment 2: Bag of Words, TF-IDF & Word2Vec

* Converted structured dataset into text format
* Implemented:

  * Bag of Words (Count Vectorizer)
  * Normalized Term Frequency (TF)
  * TF-IDF
* Generated word embeddings using **Word2Vec**
* Dataset: Car Features and MSRP (`data.csv`)

---

### 🔹 Assignment 3: Text Preprocessing & Feature Engineering

* Performed:

  * Text cleaning
  * Stopword removal
  * Lemmatization
  * Label encoding
* Generated features using **TF-IDF**
* Saved processed outputs
* Dataset: News Dataset (`News_dataset.pickle`)

---

### 🔹 Assignment 4: Transformer Model using PyTorch

* Implemented a basic Transformer architecture:

  * Embedding Layer
  * Positional Encoding
  * Transformer Encoder
* Demonstrated forward pass using sample input

---

### 🔹 Assignment 5: Morphological Analysis

* Studied morphology and word formation
* Implemented:

  * Add Table (suffix addition)
  * Delete Table (suffix removal)
* Demonstrated basic morphological transformations
